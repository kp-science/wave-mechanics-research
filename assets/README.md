# Assets

รูปภาพและสื่อที่ใช้ในเว็บไซต์

## ไฟล์ที่ใช้

| ไฟล์ | หน้าที่ | ที่ใช้ |
|------|--------|-------|
| `profile.jpg` | รูปโปรไฟล์ครู (วงกลม 128px) | `index.html` hero |

## วิธีเพิ่ม/เปลี่ยนรูปโปรไฟล์

1. ลากรูปใหม่มาไว้ที่โฟลเดอร์นี้ · ตั้งชื่อ `profile.jpg`
2. แนะนำ: รูปสี่เหลี่ยมจัตุรัส · ขนาดไม่ต่ำกว่า 400×400px
3. ใบหน้าควรอยู่ประมาณ 25-30% จากขอบบน (CSS `object-position: center 28%`)
4. Commit + push: `git add assets/profile.jpg && git commit -m "assets: update profile photo" && git push`
