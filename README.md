# วิจัยในชั้นเรียน — คลื่นกล (Wave Mechanics Research)

**หัวข้อวิจัย:** การพัฒนารูปแบบการจัดการเรียนรู้แบบสืบเสาะหาความรู้ (5E) ร่วมกับเทคนิค POE และสื่อดิจิทัลและกิจกรรมเชิงปฏิบัติ เพื่อส่งเสริมมโนทัศน์ทางฟิสิกส์และผลสัมฤทธิ์ทางการเรียน เรื่อง คลื่นกล สำหรับนักเรียนชั้นมัธยมศึกษาปีที่ 5

**ผู้วิจัย:** นายโกเมน ปาปะโถ — ครูฟิสิกส์และดาราศาสตร์
**โรงเรียน:** สตรีวิทยา · สพม. กรุงเทพมหานคร เขต 1
**รายวิชา:** ฟิสิกส์ 3 (ว30203) · ระดับ ม.5 · ปีการศึกษา 2569
**เป้าหมาย:** ยื่นขอมีวิทยฐานะ (วPA) ตามหลักเกณฑ์ ว21/2564

- **การประเมิน:** Four-tier Diagnostic Test · POE Rubric 0–3 · Traffic Light Card · Metacognitive Journal
- **เครื่องมือหลัก:**
