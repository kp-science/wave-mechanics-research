🎯 ภาพรวมระบบ

หลักการทำงาน: นักเรียนเปิดไฟล์ HTML → กรอกคำตอบ → กดปุ่ม "ส่งคำตอบ" → ข้อมูลถูกส่งไปบันทึกใน Google Sheets ของครู → ครูเปิด Sheets เพื่อวิเคราะห์ (Pivot, สูตร, Normalized Gain)

องค์ประกอบ 3 อย่าง:

  1. Google Sheet — ที่เก็บข้อมูล (Database) 1 ไฟล์ต่อ 1 เครื่องมือ หรือใช้ sheet หลายแท็บในไฟล์เดียว
  2. Apps Script (Web App) — สคริปต์ที่รับข้อมูลจาก HTML แล้วเขียนลง Sheet
  3. ไฟล์ HTML — มีปุ่ม "ส่งคำตอบ" ที่ยิง fetch POST ไปยัง URL ของ Web App

📋 ขั้นตอนติดตั้ง (ทำครั้งเดียว · ใช้เวลา 10–15 นาที)

ขั้นที่ 1 · สร้าง Google Sheet

  1. ไปที่ https://sheets.new (หรือ Google Drive → New → Google Sheets)
  2. ตั้งชื่อไฟล์: วิจัยคลื่นกล_Database_2569
  3. เปลี่ยนชื่อแท็บแรกเป็น F1_Plan1 (สำหรับ F1 แผน 1)
  4. ไม่ต้องใส่หัวตารางเอง — สคริปต์จะสร้างให้อัตโนมัติเมื่อมีข้อมูลเข้ามาครั้งแรก

ขั้นที่ 2 · เปิด Apps Script

  1. ใน Google Sheet ที่เพิ่งสร้าง → เมนู Extensions → Apps Script
  2. จะเปิดหน้าต่าง Editor ใหม่ มีไฟล์ Code.gs
  3. ลบโค้ดเก่าทั้งหมด แล้ววางโค้ดด้านล่างนี้แทน

📝 โค้ด Apps Script (Code.gs) — คัดลอกทั้งหมด

// ============================================
// Research Data Collector — วิจัยคลื่นกล
// ครูโกเมน ปาปะโถ · โรงเรียนสตรีวิทยา · 2569
// ============================================

function doPost(e) {
  try {
    const data = JSON.parse(e.postData.contents);
    const sheetName = data.sheet || 'Default';
    const ss = SpreadsheetApp.getActiveSpreadsheet();
    let sheet = ss.getSheetByName(sheetName);
    if (!sheet) sheet = ss.insertSheet(sheetName);

    const payload = data.payload || {};
    const keys = Object.keys(payload);

    // ถ้า sheet ว่าง → สร้างหัวตารางจาก keys
    if (sheet.getLastRow() === 0) {
      sheet.appendRow(['timestamp', ...keys]);
      sheet.getRange(1, 1, 1, keys.length + 1)
           .setFontWeight('bold')
           .setBackground('#1565c0')
           .setFontColor('#ffffff');
      sheet.setFrozenRows(1);
    }

    // ดึง header ปัจจุบัน แล้วเรียง value ตามลำดับ header
    const headers = sheet.getRange(1, 1, 1, sheet.getLastColumn()).getValues()[0];
    const row = headers.map(h => {
      if (h === 'timestamp') return new Date();
      return payload[h] !== undefined ? payload[h] : '';
    });

    // ถ้ามี key ใหม่ที่ยังไม่มีใน header → เพิ่มคอลัมน์
    const newKeys = keys.filter(k => !headers.includes(k));
    if (newKeys.length > 0) {
      sheet.getRange(1, headers.length + 1, 1, newKeys.length)
           .setValues([newKeys])
           .setFontWeight('bold')
           .setBackground('#1565c0')
           .setFontColor('#ffffff');
      newKeys.forEach(k => row.push(payload[k]));
    }

    sheet.appendRow(row);

    return ContentService
      .createTextOutput(JSON.stringify({status:'ok', row:sheet.getLastRow()}))
      .setMimeType(ContentService.MimeType.JSON);

  } catch (err) {
    return ContentService
      .createTextOutput(JSON.stringify({status:'error', message:err.toString()}))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

function doGet() {
  return ContentService
    .createTextOutput('Research Data Collector is running.');
}

ขั้นที่ 3 · Deploy เป็น Web App

  1. กด Save (💾) — ตั้งชื่อโปรเจกต์ วิจัยคลื่นกล_Collector
  2. กด Deploy (มุมขวาบน) → New deployment
  3. กดรูปเฟือง ⚙️ ข้างคำว่า "Select type" → เลือก Web app
  4. ตั้งค่า:
    • Description: F1 Research v1
    • Execute as: Me (komanepapato@gmail.com)
    • Who has access: Anyone ⚠️ สำคัญ ต้องเลือก Anyone ไม่งั้นนักเรียนส่งไม่ได้
  5. กด Deploy → อนุญาต (Authorize access) → เลือกบัญชี Google ของตนเอง → กด Advanced → Go to ... (unsafe) → Allow
    Google แจ้งว่า "unsafe" เพราะเป็นสคริปต์ของเราเอง ปลอดภัย
  6. จะได้ URL ลักษณะนี้ คัดลอกเก็บไว้:
https://script.google.com/macros/s/AKfycbx.........../exec

ขั้นที่ 4 · วาง URL ลงในไฟล์ HTML

  1. เปิดไฟล์ สื่อ05_F1_Fourtier_PrePost.html ด้วย text editor (เช่น Notepad, VS Code)
  2. ค้นหาบรรทัด: const API_URL = 'PASTE_YOUR_APPS_SCRIPT_URL_HERE';
  3. แทนที่ PASTE_YOUR_APPS_SCRIPT_URL_HERE ด้วย URL ที่คัดลอกจากขั้นที่ 3
  4. บันทึกไฟล์

✅ ทดสอบระบบ

  1. เปิดไฟล์ HTML ในเบราว์เซอร์ (Chrome/Edge)
  2. กรอกชื่อทดสอบ + เลือกคำตอบครบทุกข้อ
  3. กดปุ่ม "📤 ส่งคำตอบ" → ควรขึ้นข้อความ "✅ ส่งสำเร็จ"
  4. เปิด Google Sheet → ตรวจสอบว่ามีข้อมูลเข้าแท็บ F1_Plan1

🔐 ความปลอดภัย

สิ่งที่ควรรู้:
  • URL Web App เป็นความลับ — อย่าเผยแพร่สาธารณะ ใช้เฉพาะในไฟล์ HTML ที่แจกนักเรียน
  • "Anyone" คือใครก็ยิง POST มาได้ (ถ้ารู้ URL) แต่ไฟล์นักเรียนไม่ติด internet ให้เสิร์ช ก็ปลอดภัยระดับหนึ่ง
  • หากต้องการเพิ่มความปลอดภัย: เพิ่ม secret token ใน payload แล้วเช็คใน doPost
