💡 สาระสำคัญ

แสงเป็นคลื่นแม่เหล็กไฟฟ้า ที่ตามนุษย์รับรู้ได้ · ความยาวคลื่น λ ≈ 400–700 นาโนเมตร · เคลื่อนที่ในสุญญากาศด้วย c ≈ 3×10⁸ m/s
• Thomas Young (1803): ทดลองสลิตคู่ → ยืนยันแสงเป็นคลื่น (ไม่ใช่อนุภาคล้วน ตามนิวตัน) · แก้ M1.1
• เงื่อนไขแหล่งกำเนิดอาพันธ์ (coherent): f, λ เท่ากัน + เฟสต่างคงที่ · ไม่จำเป็นต้องเฟสเท่ากัน · เลเซอร์เป็นแหล่งอาพันธ์ในตัว · แก้ M1.2
• แถบสว่าง (เสริม): Δr = d sin θ = nλ (n = 0, 1, 2, ...) · n=0 คือแถบกลาง
• แถบมืด (หักล้าง): Δr = d sin θ = (n − ½)λ (n = 1, 2, 3, ...)
• มุมเล็ก θ < 10°: sin θ ≈ tan θ = x/L → d(x/L) = nλ ใช้คำนวณตำแหน่งบนฉาก
• ความต่างเฟส: Δφ = Δr · (2π/λ) · แก้ M1.4 "ระยะทาง ≠ เฟส โดยตรง · ต้องหารด้วย λ ก่อน"
🌈 สเปกตรัมคลื่นแม่เหล็กไฟฟ้า
แสงตามองเห็น = λ 400–700 nm
แสง = คลื่น EM ตัวแทน · c ≈ 3×10⁸ m/s · แก้ M1.1
🔬 สลิตคู่ของ Young
d sin θ = nλ · แถบสว่าง n=0,1,2,...
ทดลอง 1803 · ยืนยันแสง = คลื่น · สลิต = แหล่งอาพันธ์
📐 ตำแหน่งแถบบนฉาก (มุมเล็ก)
d·(x/L) = nλ · x = nλL/d
ใช้เมื่อ θ < 10° · ระยะระหว่างแถบ Δx = λL/d
📐 สรุปสมการสำคัญแผน 1 (สลิตคู่)
แถบสว่าง d sin θ = nλ n = 0, 1, 2, ...
แถบมืด d sin θ = (n − ½)λ n = 1, 2, 3, ...
มุมเล็ก d (x/L) = nλ θ < 10°
เฟส Δφ = Δr · (2π/λ) ระยะ → เฟส
⚠️ Misconception เป้าหมาย 4 ตัวในแผน 1
M1.1 ★
❌ "แสงเป็นอนุภาคล้วน เคลื่อนที่เป็นเส้นตรง · เลี้ยวเบนหรือแทรกสอดไม่ได้" [ฝังลึก · ตามนิวตัน]
✅ แสงเป็นคลื่นแม่เหล็กไฟฟ้า (Young 1803) · แทรกสอด/เลี้ยวเบนได้เช่นเดียวกับคลื่นน้ำ · ในระดับอะตอม แสดงสมบัติทั้งคลื่นและอนุภาค (wave-particle duality)
M1.2
❌ "แหล่งอาพันธ์ต้องมีเฟสเริ่มต้นเท่ากันเสมอ"
✅ เงื่อนไข coherent ต้องการแค่ f, λ เท่ากัน + เฟสต่างคงที่ · ไม่จำเป็นว่าเฟสต้องเท่ากัน · แต่เฟสต่างต้องไม่เปลี่ยนตามเวลา
M1.3
❌ "ผ่านสลิตคู่จะได้แค่ 2 จุดสว่างตรงตำแหน่งช่อง"
✅ ถ้าแสงเป็นเส้นตรงจะได้ 2 จุดจริง · แต่เพราะแสงแทรกสอด → ได้หลายแถบสว่าง-มืดสลับกัน · ระยะระหว่างแถบ Δx = λL/d
M1.4
❌ "Δr มากแสดงว่า Δφ มากตามแบบสัดส่วนตรง"
✅ ความสัมพันธ์คือ Δφ = Δr · (2π/λ) · ต้องหารด้วย λก่อน · Δr = λ → Δφ = 2π เรเดียน (1 รอบ) · ที่จำนวนเต็มของ λ จะกลับมาเฟสตรงกัน
