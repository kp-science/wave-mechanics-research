💡 สาระสำคัญ

เกรตติง (Grating) = อุปกรณ์ที่มีช่องเล็ก ๆ จำนวนมาก ขนานกัน · ระยะระหว่างช่อง d เท่ากัน · เมื่อแสงผ่านจะเกิดการเลี้ยวเบน + แทรกสอด
• d (ระยะระหว่างช่อง): d = ความกว้างของเกรตติง / จำนวนเส้นของเกรตติง · ถ้าเกรตติง N เส้นต่อ mm → d = 1/(N × 1000) ม.
• เงื่อนไขแถบสว่าง: d sin θn = nλ (n = 0, 1, 2, ...) · รูปสมการเดียวกับสลิตคู่ แต่แถบคมและสว่างกว่ามาก
• แสงขาวผ่านเกรตติง: แต่ละสีเลี้ยวเบนมุมต่างกัน (sin θ = nλ/d) → เกิดสเปกตรัม · แสงแดงเบนมาก (λ ยาว) · แสงม่วงเบนน้อย (λ สั้น) ตรงข้ามกับปริซึม (M3.2, M3.3)
• กลไก ≠ ปริซึม: เกรตติง = เลี้ยวเบน+แทรกสอด · ปริซึม = หักเหเนื่องจาก n ต่างกัน · กลไกต่างกันสิ้นเชิง (M3.1)
• ปรากฏการณ์ในชีวิตจริง: ฟองสบู่ (ฟิล์มบาง) · CD (ร่องเล็ก = เกรตติง) · โอปอ (ผลึกซิลิกาเรียงตัว) · ปีกผีเสื้อ
🌈 แสงขาวผ่านเกรตติง
d sin θ = nλ · แดงเบนมาก · ม่วงเบนน้อย
แสงแดงเบนมาก (λ ยาว) · ตรงข้ามปริซึม · แก้ M3.3
💿 CD = เกรตติงธรรมชาติ
ร่องเล็ก d ≈ 1.6 μm
ร่องเรียงเป็นวงกลม · ทำหน้าที่เกรตติง · ดูสีต่างกันตามมุมมอง
💎 โอปอ (Opal)
ผลึกซิลิกา 150–350 nm เรียงตัว
"หินนำโชค" · ออสเตรเลีย · เลี้ยวเบนเป็นสีรุ้ง
📐 สรุปสมการสำคัญแผน 3 (เกรตติง)
แถบสว่าง d sin θn = nλ n = 0, 1, 2, ...
มุมเล็ก d (x/L) = nλ θ < 10°
ทั่วไป sin θ = x / √(L² + x²) มุมใดก็ได้
d เกรตติง d = ความกว้าง / จำนวนเส้น เช่น 500 เส้น/mm → d = 2 μm

⚠️ Misconception เป้าหมาย 4 ตัวในแผน 3
M3.1 ★
❌ "เกรตติงทำงานเหมือนปริซึม · ใช้การหักเหเหมือนกัน" [ฝังลึก · เห็นสีรุ้งเหมือนกัน]
✅ กลไกต่างกัน! · ปริซึม = หักเห เนื่องจาก n(λ) ต่างกัน · เกรตติง = เลี้ยวเบน+แทรกสอด · ปริซึม: แดงเบนน้อย · เกรตติง: แดงเบนมาก ตรงข้ามกัน
M3.2
❌ "แสงขาวมีความยาวคลื่นเพียงค่าเดียว · เพิ่งแยกออกเมื่อผ่านเกรตติง"
✅ แสงขาว = แสงผสม ทุกความยาวคลื่นในช่วง 400–700 nm มารวมกันตั้งแต่ต้น · เกรตติงเพียงทำให้แต่ละ λ เลี้ยวเบนคนละมุม → เห็นแยกกัน
M3.3
❌ "ทุกสีเลี้ยวเบนมุมเดียวกัน · ขนาดของสเปกตรัมขึ้นกับ d เท่านั้น"
✅ จาก sin θ = nλ/d · มุมขึ้นกับ λ · λ มาก (แดง) → sin θ มาก → เบนมาก · λ น้อย (ม่วง) → เบนน้อย · จึงเกิดสเปกตรัม
M3.4
❌ "ฟองสบู่/CD/โอปอ ทุกอย่างมีสีจากกลไกเดียวกัน"
✅ กลไกต่างกัน 3 แบบ: ฟองสบู่ = thin film interference (สะท้อน 2 ผิว) · CD = grating (ร่องเรียงเป็นเส้น) · โอปอ = ผลึกซิลิกาเรียงเป็น 3D photonic crystal · ทั้งหมดอาศัยการแทรกสอด/เลี้ยวเบนของแสง
