ข้อ 1วัตถุสั่นอย่างง่าย ณ ปลายสุด (y = +A) ข้อใดถูกต้อง?M4.1★★
คำตอบที่ถูก
ค. ความเร็วเป็นศูนย์ ความเร่งมีค่าสูงสุดและชี้ไปทางลบ
ที่ y = +A: v = Aω cos(π/2) = 0 และ a = −ω²(+A) = −Aω² (ค่าสูงสุด ชี้ลบ/เข้าหาสมดุล) ข้อผิดพลาดที่พบบ่อยคือนักเรียนคิดว่า a = 0 ที่ปลายสุดเพราะ "หยุดนิ่งชั่วขณะ" ซึ่งผิด — ความเร็วเป็นศูนย์ไม่ได้หมายความว่าความเร่งเป็นศูนย์
❌ ความเข้าใจคลาดเคลื่อน M4.1: "หยุดชั่วขณะ → a = 0" — แรงดึงกลับยังคงกระทำอยู่เต็มที่ที่ปลายสุด
ข้อ 2วัตถุสั่นอย่างง่าย ณ จุดสมดุล (y = 0) ข้อใดถูกต้อง?M4.2
คำตอบที่ถูก
ข. ความเร็วมีค่าสูงสุด ความเร่งเป็นศูนย์
ที่ y = 0: v = ω√(A²−0²) = Aω = vmax และ a = −ω²(0) = 0 แรงดึงกลับ (F = −ky) เป็นศูนย์ที่จุดสมดุล จึงทำให้ a = 0
ข้อ 3v และ a ต่างเฟสกันเท่าใด?M4.3
คำตอบที่ถูก
ข. 90° (π/2) โดย a นำเฟส v อยู่ 90°
v = Aω cos(ωt+φ) = Aω sin(ωt+φ+π/2) และ a = −Aω² sin(ωt+φ) = Aω² cos(ωt+φ+π/2) ดังนั้น a = −ω²y นำเฟส v ไป 90°
ข้อ 4ทิศของความเร่ง a ในการสั่นอย่างง่าย?M4.4
คำตอบที่ถูก
ข. a ชี้เข้าหาจุดสมดุลเสมอ (สวนทางกับ y)
เนื่องจาก a = −ω²y เครื่องหมายลบแสดงว่า a ชี้ตรงข้ามกับ y เสมอ เมื่อ y > 0 → a < 0, เมื่อ y < 0 → a > 0
ข้อ 5y = 0.20 sin(10t) m, vmax = ?
คำตอบที่ถูก
ก. 2.0 m/s
A = 0.20 m, ω = 10 rad/s
v_max = Aω = 0.20 × 10 = 2.0 m/s
ข้อ 6y = 0.20 sin(10t) m, v ที่ y = A/2 = ?
คำตอบที่ถูก
ก. (√3/2) × vmax ≈ 1.73 m/s
v = ω√(A²−y²) = ω√(A²−(A/2)²)
= ω√(A²(1−1/4)) = ω × A√(3/4) = Aω × (√3/2)
= 2.0 × (√3/2) ≈ 1.732 m/s

ข้อ 1y = 0.20 sin(4t) m, หา v และ a ที่ t = 0.25 s
เฉลย
ωt = 4 × 0.25 = 1.00 rad
v = Aω cos(ωt) = 0.20 × 4 × cos(1.00)
= 0.80 × 0.5403 ≈ 0.432 m/s
a = −Aω² sin(ωt) = −0.20 × 16 × sin(1.00)
= −3.20 × 0.8415 ≈ −2.69 m/s²
∴ v ≈ 0.43 m/s, a ≈ −2.69 m/s² (ชี้ลบ/เข้าหาสมดุล)
ข้อ 2y = A sin(ωt), หา v เมื่อ y = A/2
เฉลย
v = ω√(A²−y²) = ω√(A²−(A/2)²)
= ω√(3A²/4) = (√3/2)Aω ≈ 0.866 Aω

ข้อ 6a ที่ y = +AM4.1★★
เฉลย
a = −ω²A (ค่าสูงสุด ชี้ทิศลบ/เข้าหาสมดุล)
a = −ω²y = −ω²(+A) = −ω²A ≠ 0 แม้วัตถุ "หยุด" ชั่วขณะ แต่แรงยังกระทำอยู่เต็มที่
⚠️ ข้อผิดพลาดที่พบ: ตอบว่า a = 0 เพราะวัตถุหยุดนิ่ง — ความเร็ว = 0 ≠ ความเร่ง = 0
ข้อ 7v กับ a ชี้ทิศเดียวกันหรือไม่?M4.4
เฉลย
ไม่เสมอไป a ชี้เข้าหาสมดุลเสมอ แต่ v ชี้ได้ทั้งสองทิศ ขึ้นกับว่าวัตถุกำลังเคลื่อนเข้าหาหรือออกจากสมดุล เช่น เมื่อ y > 0 กำลังเคลื่อนออกจากสมดุล v > 0 แต่ a < 0 → ทิศตรงข้ามกัน
ข้อ 8Challenge: y(0) = A/2 และ v(0) > 0 หา φ
เฉลย
y(0) = A sin(φ) = A/2 → sin(φ) = 1/2 → φ = π/6 หรือ 5π/6
v(0) = Aω cos(φ) > 0 → cos(φ) > 0 → φ อยู่ในควอดรันต์ที่ 1
∴ φ = π/6 (30°)
