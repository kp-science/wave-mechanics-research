🍀 Lucky Guess

T1 ถูก + T3 ผิด หรือ T1 ถูก + มั่นใจต่ำ
เดาถูก — ยังไม่มีความเข้าใจแท้จริง

❓ No Knowledge

T1 ผิด + T3 ผิด + มั่นใจต่ำทั้งคู่
ยังไม่มีความรู้ในหัวข้อนี้

4 คลังความเข้าใจผิด — SHM แผน 1–6 (18 รายการ)

รวบรวมความเข้าใจผิดที่พบบ่อยในแต่ละแผน พร้อมแนวคิดที่ถูกต้องและแนวทางแก้ไข

แผน 1 — SHM เบื้องต้น
M1.1 แอมพลิจูดกำหนดความถี่
✗ ยิ่งแกว่งไกล ยิ่งสั่นเร็ว
✓ T และ f ขึ้นกับ k, m (หรือ L, g) ไม่ใช่แอมพลิจูด เป็น isochronous
M1.2 ความเร็วสูงสุดที่ปลายทาง
✗ เร็วที่สุดที่จุดที่ไกลที่สุด เพราะ "เคลื่อนที่มาไกล"
✓ ความเร็วสูงสุดที่จุดสมดุล (x=0); ที่ปลายทาง v=0, a=max
M1.3 แรงคืนตัวคือน้ำหนัก
✗ แรงที่ทำให้สั่นคือแรงโน้มถ่วง
✓ แรงคืนตัว F = −kx สัดส่วนกับการกระจัด ชี้เข้าสมดุลเสมอ
แผน 2 — ลูกตุ้มและสปริง
M2.1 คาบลูกตุ้มขึ้นกับมวล
✗ ลูกตุ้มหนักกว่าจะแกว่งช้ากว่า
✓ T=2π√(L/g) ไม่มีมวลในสมการ มวลไม่ส่งผล (Galileo pendulum)
M2.2 สปริงแข็งกว่า → ช้ากว่า
✗ ยิ่งแข็ง ยิ่งดึงกลับช้า
✓ T=2π√(m/k) k มากขึ้น → T ลดลง (สั่นเร็วขึ้น)
M2.3 g บนดาวต่างกัน → T เหมือนกัน
✗ ลูกตุ้มสั้นเหมือนกัน คาบต้องเหมือนกัน
✓ T=2π√(L/g) — g ต่างกัน T ต่างกันทันที
แผน 3 — พลังงาน SHM
M3.1 พลังงานรวมเปลี่ยนตามตำแหน่ง
✗ ที่จุดสมดุล พลังงานรวมสูงสุด ที่ปลายทางพลังงานรวมน้อย
✓ E_total=½kA²=constant ตลอดการเคลื่อนที่ (ไม่มีการหน่วง)
M3.2 KE=PE ที่จุดสมดุล
✗ ที่ x=0 พลังงานจลน์เท่ากับพลังงานศักย์
✓ ที่ x=0 KE=E_total, PE=0; KE=PE เกิดที่ x=A/√2
แผน 4 — SHM กับวงกลม
M4.1 SHM กับ UCM ไม่เกี่ยวกัน
✗ SHM เป็นเส้นตรง UCM เป็นวงกลม ต่างกันสิ้นเชิง
✓ การฉายเงา UCM บนแกนหนึ่งคือ SHM: x=A cos(ωt+φ)
M4.2 เฟสเริ่มต้นเปลี่ยนความถี่
✗ ถ้าเริ่มไม่ที่ x=A ความถี่จะต่างออกไป
✓ φ เปลี่ยนเฉพาะจุดเริ่ม ไม่กระทบ T หรือ f
แผน 5 — Superposition
M5.1 การซ้อนทับทำลายคลื่นถาวร
✗ เมื่อคลื่นสองลูกเจอกันแล้วหักล้าง ทั้งคู่หายไปเลย
✓ คลื่นผ่านกันไม่กระทบกัน การซ้อนทับชั่วคราว คลื่นออกจากกันสมบูรณ์
M5.2 Beat frequency = ค่าเฉลี่ยความถี่สองลูก
✗ f_beat = (f₁+f₂)/2
✓ f_beat = |f₁−f₂|; ความถี่ที่ได้ยินคือ (f₁+f₂)/2
แผน 6 — การหน่วงและการสั่นพ้อง
M6.1 การหน่วงเปลี่ยนความถี่มาก
✗ น้ำทำให้ลูกตุ้มแกว่งช้าลงมากและความถี่ลดลงอย่างชัดเจน
✓ Underdamped: ω_d=√(ω₀²−(b/2m)²)≈ω₀ ความถี่เปลี่ยนน้อยมาก แอมพลิจูดต่างหากที่ลด
M6.2 การสั่นพ้อง = การสะท้อน
✗ การสั่นพ้องคือคลื่นสะท้อนกลับมาเสริมกัน เหมือนสะท้อนเสียงในถ้ำ
✓ การสั่นพ้องคือระบบรับพลังงานสะสมได้สูงสุด เมื่อ f_drive=f_nat ไม่เกี่ยวกับการสะท้อน
M6.3 การสั่นพ้อง = แรงสูงสุด
✗ ที่ความถี่สั่นพ้อง แรงที่กระทำต่อระบบจะมีค่าสูงสุด
✓ ที่สั่นพ้อง แอมพลิจูดสูงสุด (พลังงานสะสมสูงสุด) ไม่ใช่แรงสูงสุด แรงยังเป็น F₀ คงที่

5 สรุปหน่วย SHM — แผน 1–6

แผน 1
SHM เบื้องต้น
  • F = −kx (แรงคืนตัว)
  • x = A cos(ωt+φ)
  • v_max = Aω ที่ x=0
  • a_max = Aω² ที่ x=±A
  • Isochronous (A ไม่กระทบ T)
แผน 2
ลูกตุ้มและสปริง
  • ลูกตุ้ม: T=2π√(L/g)
  • สปริง: T=2π√(m/k)
  • คาบลูกตุ้ม ไม่ขึ้นกับมวล
  • k มาก → T น้อย (เร็ว)
  • g บนดาวต่าง → T ต่าง
แผน 3
พลังงาน SHM
  • E_total = ½kA² = const
  • KE = ½m(Aω)²(1−(x/A)²)
  • PE = ½kx²
  • KE=PE ที่ x=A/√2
  • v=ω√(A²−x²)
แผน 4
SHM กับวงกลม
  • UCM ฉายเงา → SHM
  • x = A cos(ωt+φ)
  • เฟสต่าง → รูปแบบต่าง
  • φ ไม่กระทบ T, f
  • ไดอะแกรม phasor
แผน 5
Superposition
  • หลักการซ้อนทับ
  • เสริมกัน / หักล้างกัน
  • Beat: f_beat=|f₁−f₂|
  • Standing wave ใน SHM
  • คลื่นไม่ทำลายกัน
แผน 6
การหน่วงและสั่นพ้อง
  • A(t)=A₀e^(−bt/2m)
  • ω_d=√(ω₀²−(b/2m)²)
  • Q=mω₀/b
  • Resonance: f_drive=f_nat
  • Under/Critical/Over
  • TMD, MRI, Tacoma

สูตรสำคัญรวม SHM

SHM พื้นฐาน
x(t) = A cos(ωt + φ)    ω = 2π/T = 2πf
v(t) = −Aω sin(ωt + φ)    a(t) = −Aω² cos(ωt + φ) = −ω²x
v = ±ω√(A² − x²)    E_total = ½kA² = ½mω²A²

ลูกตุ้ม & สปริง
T_pendulum = 2π√(L/g)    T_spring = 2π√(m/k)

Damping
A(t) = A₀ e^(−bt/2m)    ω_d = √(ω₀² − (b/2m)²)
Q = mω₀/b    Critical: b_c = 2√(km)

Resonance
สั่นพ้องเมื่อ f_drive = f_natural    → A สูงสุด
