  • POE-01 + POE Rubric → คะแนน POE รายบุคคล/กลุ่ม (เฉลี่ย ≥ 2.0)
  • TL-01 → อัตราบัตรเขียวก่อน-หลัง (เป้า ≥ 60%)
  • MJ-01 → คำถาม Hook แผน 2 · หลักฐาน Self-regulation + Transfer
  • OB-01 → คะแนนพฤติกรรม P/A · หลักฐาน วPA 3, 4, 5, 7
  • CER Live Board → เปรียบเทียบการใช้หลักฐานจาก SIM · วPA 3, 5
  • ใบงาน 1.1 + 1.2 → % ถูกต้องคำนวณ + % มาร์ค M1.1–M1.3 ถูกต้อง
