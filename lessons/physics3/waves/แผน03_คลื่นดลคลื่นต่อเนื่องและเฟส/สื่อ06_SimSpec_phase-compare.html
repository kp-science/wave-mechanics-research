🎛 Modes · 3 โหมดการใช้งาน

Mode 1 · Single Wave, Two Points (default)
แสดงคลื่นต่อเนื่อง 1 ขบวน · มี marker 2 จุด (A สีแดง, B สีน้ำเงิน) · slider เลื่อนตำแหน่ง B · แสดง Δφ ของ B เทียบ A · ด้านข้างมี "phase dial" (นาฬิกา) 2 อันโชว์มุมเฟส ณ เวลาจริง
Mode 2 · Overlay Two Waves (Elaborate challenge)
แสดงคลื่น 2 ขบวน (สีส้ม + สีฟ้า) · slider ปรับ Δφ ของคลื่น 2 (0–360°) · แสดงคลื่นผลรวม (สีเขียวเข้ม) · แสดง A รวม ณ จุดกึ่งกลาง · ปุ่ม preset: 0° 90° 180° 270°
Mode 3 · Ribbon Animation (linked to Concept Cartoon)
จำลองเชือกสั่นต่อเนื่อง · ผูกริบบิ้น 3 จุด (R แดง, B น้ำเงิน, G เขียว) ให้ user เลือกตำแหน่งได้ · แสดงการสั่นขึ้น-ลงของแต่ละริบบิ้นพร้อมกับกราฟ y–t 3 เส้นเทียบ (เฟสพร้อมกัน vs เหลื่อม)

📊 Display Elements · สิ่งที่ต้องแสดง

📈 Wave Canvas

  • แกน x: ระยะทาง · scale 0 – 2 m (ปรับตาม λ)
  • แกน y: การกระจัด · −A ถึง +A
  • คลื่น: เส้นโค้ง sine หนา 2.5 px · สี #6b46c1
  • Markers: จุด A (แดง 8 px), B (น้ำเงิน 8 px) · มี label
  • เส้นแสดง Δx: dashed · label "Δx = ___ m = ___λ"
  • Grid: แบ่งเป็น λ/4 · label λ/4, λ/2, 3λ/4, λ, ...

🕐 Phase Dial (2 อัน)

  • วงกลม radius 40 px · 12 ขีด · label 0°, 90°, 180°, 270°
  • เข็มชี้มุมเฟสปัจจุบัน · หมุนตาม animation
  • สี: A = แดง · B = น้ำเงิน
  • ด้านล่าง dial: ค่าตัวเลขเฟสสด φ_A = 45° φ_B = 225°

🧮 Readout Panel

  • Δφ (หน่วยองศา): แสดง 1 ตำแหน่งทศนิยม
  • Δφ (หน่วย rad): แสดงเป็น π/4, π/2, ... ถ้าลงตัว · ไม่งั้น 1.57 rad
  • Δx ในรูป fraction ของ λ (เช่น λ/4, 3λ/8)
  • Hint box: ข้อความอธิบายอัตโนมัติตามค่า · เช่น "สั่นพร้อมกัน (in-phase)", "สวนทางกัน (anti-phase)"

🌈 Overlay Mode (Mode 2)

  • คลื่น 1 (ส้ม #ed8936) · คลื่น 2 (ฟ้า #4299e1) · ผลรวม (เขียว #38a169 · หนากว่า)
  • แสดง A รวม ที่ x กลางจอ · ข้อความ "ผลรวม Amax = ___"
  • กรณี Δφ = 180° + A เท่ากัน: แสดง "🎯 หักล้างสมบูรณ์ (Amax = 0)"
  • กรณี Δφ = 0° + A เท่ากัน: แสดง "🎯 เสริมสมบูรณ์ (Amax = 2A)"

⚙️ Technical Spec

  • ฟอร์แมต: Standalone HTML + inline CSS + vanilla JS (ไม่ใช้ library · เปิดจาก file:// ได้)
  • Rendering: Canvas 2D API · 60 fps · auto-resize ตาม window
  • Math: y(x,t) = A·sin(2π·(x/λ − f·t) + φ₀)
  • Animation loop: requestAnimationFrame · สูตร t ++ ·
  • Touch support: slider ต้องใช้บน iPad/มือถือได้
  • Responsive: desktop ≥ 900 px, tablet 600–900 px (จัด 1 คอลัมน์), mobile < 600 px (ย่อ panel)
  • Font: Sarabun (Thai) + system sans-serif
  • Theme: Purple primary #6b46c1 · Light background #faf5ff (ตรงกับธีมแผน 3)
  • File location: lessons/physics3/waves/แผน03_.../phase-compare.html
  • Register ใน media.js: เพิ่ม entry ใน KP_PLAN_MEDIA[3] (Step 6)
