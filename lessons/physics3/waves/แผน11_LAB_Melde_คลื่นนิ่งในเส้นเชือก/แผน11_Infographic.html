💡 สาระสำคัญ

การทดลอง Melde เป็นการทดลองคลาสสิก (Franz Melde, 1859) ที่ใช้พิสูจน์คลื่นนิ่ง (Standing Wave) บนเส้นเชือกที่ยึดปลายทั้งสอง
• อัตราเร็วคลื่นในเส้นเชือก v = √(T/μ) เมื่อ T = ความตึง (N), μ = ความหนาแน่นเชิงเส้น (kg/m)
• ความถี่ฮาร์มอนิก fₙ = (n/2L)·√(T/μ) โดย n = 1, 2, 3, ... เป็นจำนวนเต็มของ antinodes
• คลื่นนิ่งเกิดจากการซ้อนทับของคลื่นไป-กลับ ที่มี f, λ เท่ากัน แต่ทิศตรงกันข้าม
• node คือจุดที่นิ่งสนิท (amplitude = 0) · antinode คือจุดที่แกว่งมากที่สุด
📐 อัตราเร็วในเส้นเชือก
v ขึ้นกับ T และ μ เท่านั้น — ไม่ขึ้นกับ f หรือ λ
แก้ M11.1 · ตึงมากกว่า → v มากกว่า (∝√T)
🎵 ฮาร์มอนิกที่เป็นไปได้
L = nλₙ/2 → λₙ = 2L/n → fₙ = nv/(2L)
n = 1 (fundamental) · n = 2, 3, ... (overtones)
🔬 เงื่อนไขการเกิด resonance
ความถี่ขับ (driver) ตรงกับ fₙ ของระบบ
แก้ M11.4 · ความถี่ผิด → ไม่เกิดคลื่นนิ่ง
vv = √(T / μ)อัตราเร็วในเส้นเชือก (m/s)
fₙfₙ = (n / 2L) × √(T / μ)ความถี่ฮาร์มอนิกที่ n
λₙλₙ = 2L / nความยาวคลื่นของฮาร์มอนิกที่ n
μμ = m / Lความหนาแน่นเชิงเส้น (kg/m)
⚠️ Misconception เป้าหมาย 4 ตัวในแผน 11
M11.1
❌ "เชือกตึงขึ้น คลื่นจะวิ่งเร็วขึ้นเชิงเส้น (T×2 → v×2)"
✅ v ∝ √T · T×4 → v×2 เท่านั้น · ต้องเพิ่ม T เป็น 4 เท่าจึงได้ v 2 เท่า
M11.2
❌ "คลื่นนิ่งคือคลื่นที่ไม่เคลื่อนที่เลย"
✅ คลื่นนิ่งเกิดจากคลื่น 2 ลูกวิ่งสวนทาง · แต่ละจุดบนเชือกยังแกว่งขึ้น-ลง · เพียงรูปแบบไม่เลื่อน
M11.3
❌ "มวลของเชือกไม่มีผลต่อ v"
✅ μ (มวลต่อความยาว) อยู่ในสูตร · เชือกหนาขึ้นที่ความตึงเดิม → v ช้าลง
M11.4
❌ "ใส่ความถี่อะไรก็ได้ จะเกิดคลื่นนิ่งเสมอ"
✅ ต้องใส่ความถี่ที่เป็น fₙ = nv/(2L) เท่านั้น · ความถี่อื่นจะได้รูปแบบมั่ว ไม่เกิด resonance

🎯 จุดประสงค์การเรียนรู้

📋ด้านความรู้ ทักษะ และเจตคติ (KPA)
🧠 ด้านความรู้ (K)
  • K11.1เขียนสมการ v = √(T/μ) และอธิบายความสัมพันธ์ระหว่าง v กับ T, μ (M11.1, M11.3) · F11 ≥ 80% Sound
  • K11.2วิเคราะห์รูปแบบฮาร์มอนิก n = 1, 2, 3 บนเส้นเชือก ระบุตำแหน่ง node/antinode (M11.2) · Rubric ≥ 3/4
🔬 ด้านทักษะกระบวนการ (P)
  • P11.1จัดการอุปกรณ์ Melde (vibrator, เชือก, ตุ้มน้ำหนัก) ปรับ T เพื่อหา resonance · Lab Rubric ≥ 2.0/3.0
  • P11.2วัด λ, f และคำนวณ v พร้อมประเมิน % error เทียบค่าทฤษฎี · เป้าหมาย %error < 10%
❤️ ด้านเจตคติ (A)
  • A11.1ทำงานเป็นทีมในห้องปฏิบัติการอย่างปลอดภัย — แบ่งบทบาทผู้วัด/ผู้บันทึก/ผู้คำนวณ · OB-11
  • A11.2วิเคราะห์ % error อย่างเป็นวิทยาศาสตร์ — ไม่ตัดข้อมูลทิ้งโดยไม่มีเหตุผล · MJ-11 ≥ 70%

  • นักเรียนรายบุคคลเลือกข้างและเขียน Predict ลง POE-11 ช่อง P พร้อมเหตุผล ≥ 2 ประโยค · 5 นาที

E2
Explore · ปฏิบัติการ Melde 3 ฐาน 32–62 นาที P11.1, P11.2 M11.1, M11.3, M11.4 👁 O 3 ฐาน × 10 นาที วPA 3, 7
  • นักเรียนกลุ่ม 4 คน หมุนเวียน 3 ฐาน — ฐาน A: เปลี่ยน T ที่ L, μ คงที่ → วัด fₙ สำหรับ n = 1, 2, 3 · ฐาน B: เปลี่ยน μ (เปลี่ยนชนิดเชือก) ที่ T, L คงที่ · ฐาน C: เปลี่ยน L (เลื่อนคลิป) ที่ T, μ คงที่
  • นักเรียนวัดและบันทึกลง POE-11 ตาราง O — รวมขั้นต่ำ 3 จุดข้อมูลต่อฐาน · ครูเดินดูทุกกลุ่ม ถามคำถามนำให้คิด ไม่เฉลย
  • นักเรียนสร้างกราฟ v² vs T (ฐาน A) บนกระดาษกราฟ → ตรวจสอบว่าได้เส้นตรงผ่าน origin หรือไม่
E3
Explain · CER Live Board 62–77 นาที K11.1, K11.2 ✍️ E วPA 3, 5, 6
  • แต่ละกลุ่มนำเสนอ Claim-Evidence-Reasoning จากฐานที่ได้ บนกระดาน Live Board (Padlet) · เน้นเรื่อง slope ของกราฟ v² vs T = 1/μ
  • นักเรียนและครูร่วมกันอภิปรายเปรียบเทียบ Predict (P) กับ Observation (O) · เขียน Explain (E) ใน POE-11 ช่อง E
  • ครูสรุปและเชื่อมโยงสมการ v = √(T/μ) · fₙ = nv/(2L) · ย้ำว่าเชือกหนัก (μ สูง) → v ต่ำ (แก้ M11.3)

  • นักเรียนทำใบงาน 11.2 Spot the Error — มาร์ค misconception ใน 4 ข้อความผิด · 2 ข้อแฝง M11.1 · 2 ข้อแฝง M11.4
