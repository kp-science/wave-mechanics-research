💡 สาระสำคัญ: คลื่นนิ่ง (Standing Wave) เกิดจากคลื่นสองขบวนที่มี f, λ, A เท่ากันเดินทางสวนทางกัน (ส่วนใหญ่คือคลื่นตกกระทบ + คลื่นสะท้อน) · ซ้อนทับกันตามหลักการซ้อนทับ (แผน 4) → เกิดตำแหน่งคงที่: N (Node · บัพ) = จุดไม่สั่น (สวนเฟส) · A (Antinode · ปฏิบัพ) = จุดสั่นแรงสุด (เฟสเดียวกัน) · ระยะระหว่าง N ที่อยู่ติดกัน = λ/2 · บนเส้นเชือกสองปลายตรึง: L = n·λ/2 → fₙ = (n/2L)√(T/μ) เกิด harmonic n = 1, 2, 3, ... · บูรณาการ สะท้อน + ซ้อนทับ + หักเห + แทรกสอด + เลี้ยวเบน
🎼Harmonic Modes · n = 1, 2, 3 (เชือกสองปลายตรึง L คงที่)
ระยะ N–N ติดกัน = λ/2 · fₙ = n·f₁ = (n/2L)√(T/μ) · สูตรลัด Lab 37: λ = 2d (d = ระยะ Node 2 จุดใกล้กัน)

⚠️Misconceptions ที่จะชน (4 ข้อ)
M9.1 ★ MWCS
❌ "คลื่นนิ่ง = คลื่นที่หยุดนิ่ง ไม่เคลื่อนไหวแล้ว"
✅ คลื่น 2 ขบวนสวนทางยังเคลื่อนที่อยู่ · แค่ตำแหน่ง A/N คงที่ · พลังงานยังอยู่ในระบบ
M9.2 CL
❌ "N (Node) คือจุดที่สั่นมากที่สุด"
✅ กลับกัน · A (Antinode) = จุดสั่นแรงสุด · N (Node) = จุดไม่สั่นเลย (2 คลื่นหักล้างตลอดเวลา)
M9.3 ★ CL
❌ "คลื่นขบวนเดียวก็ทำให้เกิดคลื่นนิ่งได้"
✅ ต้องมีขั้นต่ำ 2 ขบวน · ทิศตรงข้าม · λ, f, A เท่ากัน (ปกติเป็น incident + reflected)
M9.4 CL
❌ "ความถี่ธรรมชาติของเชือกมีค่าเดียว"
✅ มีหลายค่า (harmonics) · fₙ = n·f₁ = (n/2L)√(T/μ) · n = 1, 2, 3, ...
🚀Engage · Hook "🎬 คลิปไมโครเวฟจุดร้อน" (Microwave Standing Wave)

🎥 คลิป: นำชีสเยิ้ม/มาร์ชเมลโลว์เรียงในจานไมโครเวฟ (ถอดจานหมุนออก) · อุ่น 30 วินาที · สังเกตจุดหลอมสลับจุดไม่หลอมเป็นระยะเท่ากัน

❓ คำถาม: (1) ทำไมมาร์ชเมลโลว์บางจุดเยิ้ม บางจุดไม่เยิ้ม? · (2) ระยะระหว่างจุดร้อน = λ/2 · ถ้าวัดได้ 6 cm → f ของไมโครเวฟ = ? (c = 3×10⁸ m/s)

→ Hook สั้น ทรงพลัง · เชื่อม K9.1 (A=ร้อน, N=ไม่ร้อน) + K9.2 (f, λ) + K9.3 (สะท้อนผนัง+ซ้อนทับ) ครบในคลิปเดียว · นักเรียนทำ POE-P ก่อนเปิด Lab 37
