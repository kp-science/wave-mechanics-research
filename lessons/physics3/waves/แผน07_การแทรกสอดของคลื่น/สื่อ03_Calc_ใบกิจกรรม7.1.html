แผน 7ขั้น E4 4 ข้อ ไล่ระดับ · ประเมิน K7.1, K7.2, P7.1, P7.2 · Rubric 0–4 · เวลา 15 นาที
🎯 โจทย์ฝึก: ประยุกต์เงื่อนไขการแทรกสอด · ปฏิบัพ A: Δr = nλ (n = 0,1,2,...) เสริม y = 2A (ถ้า A เท่า) · บัพ N: Δr = (n+½)λ หักล้าง y = 0 (ถ้า A เท่า) · เงื่อนไข coherent: f, λ เท่า + เฟสสัมพันธ์คงที่ (A ไม่ต้องเท่า) · แสดงวิธีทุกข้อ
1 วาดรูปแบบการแทรกสอด 2 แหล่ง · S₁S₂ ห่างกัน 3λ ★ ง่าย
แหล่งกำเนิดอาพันธ์ S₁ และ S₂ · เฟสตรงกัน · A เท่ากัน · λ = 2 cm · ระยะห่าง S₁S₂ = 3λ = 6 cm
✏️ งานของนักเรียน — วาดเพิ่มบนภาพ:
① วาดวงกลมหน้าคลื่นจาก S₁ และ S₂ (ห่างกัน λ = 2 cm = 40 px ต่อวง · อย่างน้อย 4 วง)
② ระบุแนวปฏิบัพ A (เส้นที่ยอดพบยอด — วงเส้นเต็ม·พบเส้นเต็ม)
③ ระบุแนวบัพ N (เส้นที่ยอดพบท้อง — เส้นเต็ม·พบเส้นประ — ใช้ภาพที่ยอด/ท้องสลับ) · ใช้สัญลักษณ์ A และ N
④ นับและจดจำนวน แนว A กี่แนว และ แนว N กี่แนว ระหว่าง S₁ ถึง S₂
💡 จำนวนแนวบัพ N ระหว่าง 2 แหล่ง = 2(S₁S₂/λ) = 2(3) = 6 แนว · จำนวนปฏิบัพ A = 2(S₁S₂/λ) + 1 = 7 แนว
คำถาม:
(ก) จำนวนแนว A ที่นับได้ระหว่าง S₁S₂ = ______ แนว   (ข) จำนวนแนว N = ______ แนว   (ค) แนวกึ่งกลาง (Δr=0) เป็น A หรือ N?
🔑 แนวตอบ: (ก) A = 7 แนว (n = 0, ±1, ±2, ±3) · (ข) N = 6 แนว (n+½ = ±0.5, ±1.5, ±2.5) · (ค) แนวกึ่งกลาง A เพราะ Δr = 0 = 0·λ ตรงเงื่อนไข nλ พอดี (n=0 → ปฏิบัพ A) · แก้ M7.1
2 คำนวณ |S₁P − S₂P| · ตัดสินจุด P เป็น A หรือ N ★★ กลาง
แหล่งกำเนิด S₁ และ S₂ เฟสตรงกัน · A เท่ากัน · λ = 4 cm · จุด P วัดได้ S₁P = 20 cm · S₂P = 14 cm
เงื่อนไข: Δr = nλ → ปฏิบัพ A (เสริม) · Δr = (n+½)λ → บัพ N (หักล้าง)
คำถาม: (ก) คำนวณ Δr = |S₁P − S₂P| · (ข) เทียบ Δr/λ ได้เท่าใด · (ค) จุด P เป็น A หรือ N · ค่า n เป็นเท่าใด · (ง) ถ้า A₁ = A₂ = 5 cm · การกระจัดรวมที่ P เป็นเท่าใด?
✍️ กระดานแสดงวิธีทำ:
🔑 แนวตอบ: (ก) Δr = |20 − 14| = 6 cm · (ข) Δr/λ = 6/4 = 1.5 = (1+½) · (ค) P เป็นแนวบัพ N ที่ n = 1 (ตรงเงื่อนไข Δr = (n+½)λ พอดี) · (ง) ที่บัพ A เท่ากัน → y รวม = 0 (หักล้างสมบูรณ์ตลอดเวลา · อนุภาคยังอยู่ ไม่หาย — แก้ M7.2)
3 เลื่อน source separation d → จำนวน A เพิ่ม/ลด · กราฟ ★★ กลาง
ให้ λ = 2 cm คงที่ · เลื่อนระยะห่าง 2 แหล่ง d = S₁S₂ ตามค่าในตาราง · จำนวนแนวปฏิบัพ A ระหว่าง S₁ ถึง S₂ คำนวณจาก: N_A = 2⌊d/λ⌋ + 1
สูตร: ถ้า d = mλ พอดี → จำนวน A = 2m + 1 (รวมแนวกึ่งกลาง) · จำนวน N = 2m

คำถาม:
(ก) เติมตารางให้ครบ   (ข) ความชันของกราฟ N_A vs d/λ ≈ ______   (ค) เมื่อ d เพิ่ม จำนวน A เพิ่มหรือลด · ทำไม?
🔑 แนวตอบ: (ก) d = 2,4,6,8,10 cm → d/λ = 1,2,3,4,5 → A = 3,5,7,9,11 · N = 2,4,6,8,10 · (ข) slope ≈ 2 (ทุก λ ที่เพิ่ม → A เพิ่ม 2 แนว) · (ค) เพิ่ม · เพราะ d ใหญ่ขึ้น → จำนวน path differences ที่ตรงเงื่อนไข nλ มากขึ้น
4 Path difference Δr = 2.5λ → คำตอบ N (n=2) ★★★ ท้าทาย
แหล่งกำเนิด S₁ (A₁ = 4 cm) และ S₂ (A₂ = 4 cm) · เฟสตรงกัน · f = 5 Hz · v = 20 cm/s · ที่จุด Q วัดได้ |S₁Q − S₂Q| = 10 cm
v = fλ → λ = v/f · เงื่อนไข A: Δr = nλ · เงื่อนไข N: Δr = (n+½)λ
คำถาม:
(ก) คำนวณ λ · (ข) คำนวณ Δr/λ · (ค) Q เป็น A หรือ N · n = ? · (ง) การกระจัดรวมที่ Q (เมื่อ A เท่ากัน) · (จ) ถ้าเปลี่ยน A₂ = 2 cm (ต่างกัน) ที่จุด Q ใหม่นี้ การกระจัดรวมเป็นเท่าใด · เกิดการแทรกสอดได้ไหม (M7.3)?
✍️ กระดานแสดงวิธีทำ:
🔑 แนวตอบ: (ก) λ = v/f = 20/5 = 4 cm · (ข) Δr/λ = 10/4 = 2.5 = (2+½) · (ค) Q เป็นแนวบัพ N ที่ n = 2 (ตรงเงื่อนไข Δr = (n+½)λ พอดี) · (ง) A เท่ากัน (4 cm ทั้งคู่) · ที่บัพ → y รวม = 0 (หักล้างสมบูรณ์) · อนุภาคยังอยู่ ไม่หาย (แก้ M7.2) · (จ) ถ้า A₁ = 4 · A₂ = 2 · ที่บัพ N · y₁ = +4 · y₂ = −2 → y รวม = +2 cm (หักล้างไม่สมบูรณ์) · เกิดการแทรกสอดได้ · เงื่อนไข coherent ต้องการแค่ f, λ เท่ากัน + เฟสคงที่ ไม่ต้อง A เท่า (แก้ M7.3)
