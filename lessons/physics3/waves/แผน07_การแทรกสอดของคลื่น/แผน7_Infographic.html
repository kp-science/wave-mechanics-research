💡 สาระสำคัญ

เมื่อคลื่นจากแหล่งกำเนิดอาพันธ์ (coherent) 2 แหล่ง มาพบกัน · เกิดการแทรกสอด (Interference) · ตามหลักการซ้อนทับ y = y₁ + y₂
• เงื่อนไขปฏิบัพ A: Δr = nλ (n = 0, 1, 2,...) · ยอดพบยอด · เสริมกัน · y รวม สูงสุด = A₁ + A₂ (= 2A ถ้า A เท่า)
• เงื่อนไขบัพ N: Δr = (n+½)λ · ยอดพบท้อง · หักล้าง · y รวม = |A₁ − A₂| (= 0 ถ้า A เท่า) (แก้ M7.1, M7.4)
• บัพ N · อนุภาคยังอยู่ · การกระจัด = 0 ตลอดเวลาเพราะหักล้างชั่วขณะต่อเนื่อง · พลังงานถูกส่งต่อไปสะสมที่ปฏิบัพ A (แก้ M7.2)
• เงื่อนไข coherent: f, λ เท่ากัน + เฟสสัมพันธ์คงที่ · A ไม่ต้องเท่ากัน · ถ้า A ต่างกันยังเกิดการแทรกสอดได้แต่หักล้างไม่สมบูรณ์ (แก้ M7.3)
🔆 ปฏิบัพ A · เสริม
Δr = nλ → A รวม สูงสุด
ยอดพบยอด · เสริมกัน · แก้ M7.1 + M7.4
🌑 บัพ N · หักล้าง
Δr = (n+½)λ → y รวม = 0
ยอดพบท้อง · หักล้าง · อนุภาคยังอยู่ ไม่หาย · แก้ M7.2
📐 รูปแบบไฮเพอร์โบล่า
แนว A และ N สลับกัน · ขึ้นกับ Δr
รูปแบบคงที่ · A ไม่ต้องเท่ากัน · เงื่อนไข coherent · แก้ M7.3
⚠️ Misconception เป้าหมาย 4 ตัวในแผน 7
M7.1 ★
❌ "Δr = nλ คือแนวบัพ N" [★ ฝังลึกที่สุด]
✅ Δr = nλ คือแนวปฏิบัพ A (เสริม) · แนวบัพ N ใช้เงื่อนไข Δr = (n+½)λ
M7.2
❌ "บัพ = อนุภาคหายถาวร · น้ำหายไป"
✅ บัพคือจุดที่การกระจัด = 0 ตลอดเวลาเพราะหักล้างกันชั่วขณะต่อเนื่อง · อนุภาคยังอยู่ · พลังงานส่งต่อไปสะสมที่ปฏิบัพ
M7.3
❌ "ต้องใช้ A เท่ากันเท่านั้นถึงจะแทรกสอดได้"
✅ ใช้ A ต่างกันก็แทรกสอดได้ · เงื่อนไข coherent ต้องการ f, λ เท่ากัน + เฟสสัมพันธ์คงที่ · เพียงแต่หักล้างที่บัพไม่สมบูรณ์
M7.4
❌ "สับสน A กับ N บนรูป"
✅ A = Antinode (ปฏิบัพ · A สูงสุด) · N = Node (บัพ · A ศูนย์) · เป็นชื่อตรงข้ามกัน · จดจำผ่านสีและสัญลักษณ์ในรูป
