## โครงสร้างแต่ละแผน
ทุกแผนใช้รูปแบบเดียวกัน เพื่อง่ายต่อการ submit วPA และตรวจประกัน:
1. **สาระสำคัญ / ผังความสัมพันธ์**
2. **มาตรฐาน/ตัวชี้วัด/จุดประสงค์การเรียนรู้ (K-P-A)**
3. **สมรรถนะสำคัญและคุณลักษณะ**
4. **สาระการเรียนรู้** (เนื้อหาที่ครูต้องสอน)
5. **กระบวนการเรียนรู้แบบ 5E** (Engage → Explore → Explain → Elaborate → Evaluate)
   - แทรก **Motivation Layer** (Curiosity Gap, Variable Reward, Flow, SDT)
   - แทรก **Misconception Probe** ทุกแผน
6. **สื่อ/แหล่งเรียนรู้**
7. **การวัดและประเมินผล** (พร้อม rubric / Four-tier / Formative)
8. **บันทึกผลหลังจัดการเรียนรู้** (ตามฟอร์ม วPA)

## หลักการออกแบบ
- **5E + POE** เป็นแกนหลัก
- **Two-tier / Four-tier diagnostic** ในแผน 1, 5, 8 (ตามจุดเปลี่ยนสำคัญ)
- **Concept Cartoon** ทุกแผนใน Engage (10 นาทีแรก)
- **Mini-Lab / Simulation** ทุกแผนใน Explore (≥ 15 นาที/คาบ)
- **CER (Claim-Evidence-Reasoning)** ใน Explain
- **โจทย์ Real-life** ใน Elaborate (ดนตรีไทย, เครื่อง Ultrasound, ดอปเพลอร์รถพยาบาล ฯลฯ)
- **Exit Ticket** ใน Evaluate ทุกคาบที่ 2
