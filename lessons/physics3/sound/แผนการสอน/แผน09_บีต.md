# แผนการจัดการเรียนรู้ที่ 9
## เรื่อง บีต (Beat)

**หน่วยที่** 12 เสียง · **เวลา** 2 คาบ × 50 นาที · **ระดับชั้น** ม.5
**สอดคล้องกับ** สสวท. ฟิสิกส์ เล่ม 4 บท 12.3 (จุดประสงค์ข้อ 13)

---

### 1. สาระสำคัญ
**บีต (Beat)** เป็นปรากฏการณ์การแทรกสอดเชิงเวลาของคลื่นเสียง 2 ขบวนที่มีความถี่ใกล้เคียงกัน (Δf เล็ก) ผลรวมทำให้แอมพลิจูดเปลี่ยนแปลงช้า ๆ สลับดัง–ค่อย ผู้ฟังได้ยินเสียง "เต้น" เป็นจังหวะ **ความถี่บีต f_b = |f₁ − f₂|** ใช้ในการปรับเสียงเครื่องดนตรี (Tuning): นักดนตรีค่อย ๆ ปรับสายให้ f_b → 0 แสดงว่าสอง f ตรงกัน

### 2. มาตรฐาน/ตัวชี้วัด/จุดประสงค์
**ตัวชี้วัด ม.5/7** ทดลองและอธิบายการเกิดบีต
- **K:** อธิบายการเกิดบีตและคำนวณความถี่บีต f_b = |f₁ − f₂| ได้
- **P:** ทดลองสังเกตการเกิดบีตและวัดความถี่บีต
- **P:** ประยุกต์บีตในการปรับเสียงเครื่องดนตรี
- **A:** เห็นคุณค่าของการเชื่อมโยงฟิสิกส์กับศิลปะการดนตรี

### 3. สมรรถนะ/คุณลักษณะ
- คิดวิเคราะห์ · แก้ปัญหา · เชื่อมข้ามศาสตร์

### 4. สาระการเรียนรู้
1. **เงื่อนไขเกิดบีต:** Δf ≤ ~15 Hz (ถ้าต่างมากเกิน จะแยกเป็น 2 ความถี่)
2. **สมการ:** f_b = |f₁ − f₂|
3. **คาบบีต:** T_b = 1/f_b → ระยะเวลาจากดังที่สุดถึงดังที่สุดถัดไป
4. **การใช้งาน:** การปรับเสียงเครื่องดนตรี (ใช้ส้อมเสียงอ้างอิง 440 Hz กับสายกีตาร์) · เครื่องวัดความเร็ว/Sensor (acoustic doppler) · ในเครื่องบินตรวจจับ propeller imbalance
5. **เชื่อมกับ Superposition:** y = 2A·cos(2π·(f_b/2)·t) · cos(2π·f_avg·t) → envelope ความถี่ f_b/2 แต่ "เสียงดัง" 2 ครั้ง/รอบ → ผู้ฟังรับรู้ที่ f_b

---

### 5. กระบวนการ 5E

#### คาบที่ 1 — Engage + Explore + Explain

**Engage (10 นาที)**
- 🎬 คลิป "Guitar tuning by beat" + คลิปนักดนตรีจูนเปียโน
- 🎵 ครูเล่นเสียง 2 ส้อมเสียง: 440 Hz + 442 Hz พร้อมกัน → ได้ยินเสียง "เต้น" 2 ครั้ง/วินาที
- ❓ Curiosity Gap: *"ความถี่ 440 และ 442 Hz ห่างกันแค่ 2 Hz ทำไมหูเราได้ยินเสียงเต้นชัดเจน?"*
- 📋 **POE (Predict-Observe-Explain):**
  - **Predict:** ถ้าเล่น 2 เสียงพร้อมกัน 440 + 444 Hz จะได้ยินกี่ครั้ง/วินาที?
  - **Observe:** ครูเล่น → นักเรียนนับ
  - **Explain:** สมการ f_b

**Explore (30 นาที) — ทดลอง Beat (กิจกรรม 2.7)**
- **อุปกรณ์:** Tone Generator 2 ตัว (แอปบนมือถือคนละเครื่อง) หรือ Audacity 2 track
- **กิจกรรม:**
  - ขั้น 1: ปล่อย f₁ = 500 Hz · f₂ = 504 Hz → ฟัง + นับ beat ใน 10 วินาที
  - ขั้น 2: เปลี่ยน f₂ = 505, 508, 510, 515, 520 Hz → จำนวน beat/วินาที
  - ขั้น 3: สังเกตว่าถ้า Δf > 15 Hz ฟังเริ่มไม่เป็น beat แต่เป็น 2 ความถี่
  - ขั้น 4: ปล่อย f₂ = 500 Hz เท่ากัน → ฟัง beat = 0
- 📋 บันทึก: f₁ | f₂ | จำนวน beat / 10 s | f_b คำนวณ | f_b ทฤษฎี
- **Misconception Probe (5 นาที):**
  1. ถ้า 2 ลำโพงเล่นความถี่เท่ากัน เกิดบีตไหม
  2. f_b = f₁ + f₂ หรือ |f₁ − f₂|
  3. ทำไมถ้า Δf ใหญ่ จึงไม่ได้ยินเป็นบีต

**Explain (10 นาที)**
- ครูพิสูจน์เชิงคุณภาพด้วย superposition: y₁ + y₂ → 2cos(2π·f_avg t) · cos(2π·(Δf/2) t)
- วาดกราฟ envelope บนกระดาน
- 📝 **ตัวอย่าง 12.9 จากตำรา:** นักดนตรีเทียบเสียงกีตาร์กับหลอด 392 Hz · ได้ยินดัง–ค่อย 6 ครั้งใน 2 วินาที
  - f_b = 3 Hz → f_guitar = 392 ± 3 = 389 หรือ 395 Hz

#### คาบที่ 2 — Elaborate + Evaluate + Bridge

**Elaborate (35 นาที)**
- 📋 **ใบงาน 3.2 — Math Builder:**
  - **A:** 440 Hz + 444 Hz → f_b = ?
  - **A:** ส้อมเสียง 256 Hz กับสายกีตาร์ ได้ยิน beat 4 ครั้ง/วินาที → สายกีตาร์ = ?
  - **B:** ส้อมเสียง A + B ให้ beat 6 Hz · ส้อมเสียง A + C ให้ beat 4 Hz · ถ้า A = 440 Hz, B > A, C < A หา B และ C
  - **B:** ผู้ฟังได้ยิน beat 5 Hz เมื่อเปิด 2 ลำโพงพร้อมกัน · ลำโพง 1 = 1000 Hz · ลำโพง 2 = ? (2 คำตอบ)
  - **C — ตัวอย่าง 12.9 ขยาย:** ถ้านักดนตรีขันสายให้แน่นขึ้น แล้วได้ยิน beat เพิ่ม → ก่อนหน้านี้สาย f สูง/ต่ำกว่าหลอด
  - **C — สอวน. lite:** ส้อมเสียง 3 อัน ก, ข, ค · f_ก > f_ค · ก+ข ให้ beat 3 Hz · ข+ค ให้ beat 4 Hz · f_ข = 640 Hz · หา f_ก
- 🎯 **กิจกรรมจริง — Tune the Guitar:** ใช้กีตาร์ห้องดนตรีหรือคลิป
  - ครูปล่อยเสียง 440 Hz · นักเรียนปรับสายเอกของกีตาร์จนได้ beat ≈ 0
  - สังเกตทักษะการฟัง: ปรับขึ้น–ปรับลง วิธีไหนเร็วกว่า
- 🪙 Variable Reward: กลุ่มที่จูนได้ใกล้สุด (เครื่องวัดความถี่ยืนยัน) ได้ดาวพิเศษ

**Evaluate (10 นาที)**
- 📝 **Exit Ticket (4 ข้อ):**
  1. สูตร f_b คือ
  2. ส้อมเสียง 500 และ 502 Hz เกิด beat กี่ Hz
  3. ถ้า f_b = 0 หมายความว่าอย่างไร
  4. ทำไม Δf > 15 Hz ไม่ได้ยินเป็น beat
- 🚦 Traffic Light + Journal

**Bridge:** ครูถาม "**แล้วถ้าผู้ฟังหรือแหล่งกำเนิดเคลื่อนที่ ความถี่จะเปลี่ยนยังไง?**" → ใบ้ดอปเพลอร์
