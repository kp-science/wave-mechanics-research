# แผนการจัดการเรียนรู้ที่ 8
## เรื่อง คลื่นนิ่งของเสียงและการสั่นพ้องของอากาศในท่อปลายปิด

**หน่วยที่** 12 เสียง · **เวลา** 2 คาบ × 50 นาที · **ระดับชั้น** ม.5
**สอดคล้องกับ** สสวท. ฟิสิกส์ เล่ม 4 บท 12.3 (จุดประสงค์ข้อ 10, 11, 12)

---

### 1. สาระสำคัญ
**คลื่นนิ่งของเสียง** เกิดจากคลื่นเสียง 2 ขบวนที่มีความถี่เท่ากันและเคลื่อนที่สวนทาง รวมกัน (เช่น เสียงจากลำโพง + เสียงสะท้อนจากผนัง) เกิดบริเวณคงที่เรียก **ปฏิบัพ (antinode)** และ **บัพ (node)** เมื่ออากาศในท่อปลายปิดด้านหนึ่งสั่นด้วยความถี่ตรงกับความถี่ธรรมชาติ จะเกิด **การสั่นพ้อง (Resonance)** ลำอากาศได้ยินเสียงดังที่สุด ความยาวลำอากาศที่เกิดเสียงดังที่สุดสัมพันธ์กับ λ/4, 3λ/4, 5λ/4, ... ความถี่ธรรมชาติของท่อปลายปิดข้างเดียว: **f_n = nv/(4L)** เมื่อ n = 1, 3, 5, ... (ฮาร์มอนิกคี่)

### 2. มาตรฐาน/ตัวชี้วัด/จุดประสงค์
**ตัวชี้วัด ม.5/7** ทดลองและอธิบายคลื่นนิ่ง การสั่นพ้องของอากาศในท่อปลายปิด
- **K:** อธิบายการเกิดคลื่นนิ่งของเสียงและตำแหน่งของ node/antinode ในท่อปลายปิด
- **K:** คำนวณความถี่ธรรมชาติ ความยาวคลื่น และอัตราเร็วเสียงจากข้อมูลการสั่นพ้อง
- **P:** ทำการทดลองการสั่นพ้องในท่อปลายปิดและวัดความยาวคลื่นเสียงในอากาศ
- **A:** มีความรอบคอบในการบันทึกและวิเคราะห์ข้อมูลเชิงปริมาณ

### 3. สมรรถนะ/คุณลักษณะ
- การคิดวิเคราะห์ · ทักษะการทดลอง · ความรอบคอบ

### 4. สาระการเรียนรู้
1. **คลื่นนิ่งของเสียง:** เกิดจากการซ้อนทับของคลื่นเสียง 2 ขบวนสวนทาง ความถี่และแอมพลิจูดเท่ากัน
2. **บัพ (Node) / ปฏิบัพ (Antinode)** ในเสียง:
   - ในคลื่นการกระจัด: ปลายปิด = node, ปลายเปิด = antinode
   - ในคลื่นความดัน: สลับกัน (ปลายปิด = antinode ความดัน)
3. **ท่อปลายปิดข้างเดียว (L):**
   - Mode 1: L = λ₁/4 → f₁ = v/(4L)
   - Mode 2: L = 3λ₃/4 → f₃ = 3v/(4L)
   - Mode 3: L = 5λ₅/4 → f₅ = 5v/(4L)
   - **f_n = nv/(4L), n = 1, 3, 5, ...** (ฮาร์มอนิกคี่)
4. **การทดลอง:** เปลี่ยนความยาวลำอากาศโดยเลื่อนลูกสูบ → จุดที่ดังที่สุดคือจุดสั่นพ้อง
5. **ระยะห่างระหว่างจุดสั่นพ้อง 2 ตำแหน่งติดกัน = λ/2**

### 5. กระบวนการ 5E

#### คาบที่ 1 — Engage + Explore (กิจกรรมหลัก)

**Engage (10 นาที)**
- 🎬 คลิป "แก้วไวน์แตกเพราะเสียง" + "Tacoma Narrows Bridge" + "ขลุ่ย/ขวด ตัวเล็กตัวใหญ่ ทำไมเสียงต่างกัน"
- ❓ Curiosity Gap: *"ทำไมเป่าขวดเปล่าได้เสียง? เป่าขวดน้ำเต็มได้เสียงต่างไป?"*
- 📋 **Concept Cartoon "เป่าขวด":** 3 ตัวการ์ตูนเถียงว่าน้ำในขวดมีผลกับเสียง

**Explore (40 นาที) — ทดลองสั่นพ้องในท่อ (กิจกรรม 2.5–2.6)**
- **Misconception Probe (5 นาที):**
  1. คลื่นนิ่งคืออะไร แตกต่างจากคลื่นเดินทางอย่างไร
  2. ท่อยาวขึ้น เสียงสูงขึ้นหรือต่ำลง + เพราะ
- **กิจกรรม 2.5 (Resonance Tube) — 25 นาที:**
  - อุปกรณ์: ท่อสั่นพ้องพร้อมลูกสูบ · ลำโพง · เครื่องกำเนิดสัญญาณ (Tone generator)
  - ขั้นตอน:
    1. ตั้งความถี่ลำโพง **f = 1 kHz**
    2. วางลำโพงใกล้ปลายเปิด เลื่อนลูกสูบช้า ๆ จาก L = 0
    3. จดตำแหน่ง L₁ (เสียงดังที่สุดครั้งแรก) → L = λ/4
    4. เลื่อนต่อ จด L₂ (ดังที่สุดครั้งที่ 2) → L = 3λ/4
    5. หา **λ = 2(L₂ − L₁)**
    6. คำนวณ **v = fλ** เปรียบเทียบกับ v = 331 + 0.6·T_C
  - ทำซ้ำที่ f = 2 kHz และ 3 kHz
- 📋 บันทึก: ตาราง f | L₁ | L₂ | λ | v · กราฟ L vs f
- 💡 **คาดหวัง:** v ที่ได้ใกล้เคียง 340–345 m/s ที่อุณหภูมิห้อง

#### คาบที่ 2 — Explain + Elaborate + Evaluate

**Explain (20 นาที)**
- ครูวาดแผนภาพ standing wave ในท่อปลายปิด: mode 1, 3, 5
  - Mode 1: λ₁ = 4L → f₁ = v/(4L)
  - Mode 3: λ₃ = 4L/3 → f₃ = 3v/(4L)
  - Mode 5: λ₅ = 4L/5 → f₅ = 5v/(4L)
- เน้น "ปลายปิด = node การกระจัด" / "ปลายเปิด = antinode การกระจัด"
- เชื่อมกับการทดลอง: ที่ f คงที่ การเลื่อนลูกสูบ = เปลี่ยน L → ปรับให้ L = (2k−1)λ/4 → เสียงดังที่สุด
- ตอบกลับ Concept Cartoon: ขวดที่น้ำมากขึ้น = L อากาศสั้นลง → f สูงขึ้น

**Elaborate (20 นาที) — Math Builder**
- 📋 **ใบงาน 3.1:**
  - **A:** ท่อปลายปิดยาว 0.5 m, v=340 m/s → f₁ = ? f₃ = ?
  - **A:** ในการทดลอง f = 500 Hz, L₁ = 17 cm, L₂ = 51 cm → λ = ? v = ?
  - **B:** ท่อยาว L ปลายปิด เกิดสั่นพ้องที่ 200, 600, 1000 Hz → L = ? (ที่ v = 340)
  - **B:** คลื่นนิ่งในท่อปลายปิด f₁ = 256 Hz (โน้ต C4), v=340 → L = ?
  - **C — สอวน. lite:** เปรียบเทียบความถี่ของท่อปลายปิดข้างเดียวกับท่อปลายเปิด 2 ข้างที่ยาวเท่ากัน
- 🎯 **Real-life:** ท่อแอร์ในรถยนต์บางครั้งดังเสียงคำราม (drone) ที่ความเร็วคงที่ — เกิดจาก resonance ลำอากาศในระบบไอเสีย — ผู้ผลิตใช้ "Helmholtz resonator" แก้ไข

**Evaluate (10 นาที)**
- 📝 **Exit Ticket (5 ข้อ):**
  1. ในท่อปลายปิด ปลายปิดเป็น node ของ … (การกระจัด)
  2. ท่อปลายปิดยาว 25 cm ความถี่ธรรมชาติต่ำสุดเท่าไร (v=340)
  3. ความถี่ธรรมชาติของท่อปลายปิดมีเฉพาะฮาร์มอนิก … (คี่)
  4. ระยะระหว่าง L₁ กับ L₂ ในการทดลอง = … ของ λ (1/2)
  5. คำนวณ v จาก f=1000 Hz, L₁=8.6 cm, L₂=25.8 cm
