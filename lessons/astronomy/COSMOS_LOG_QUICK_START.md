## 📤 วิธีใช้

### ครั้งแรก (set up Grok Project)
1. เปิด Grok → สร้าง Project "COSMOS LOG"
2. **Project Instructions:** paste section "WORKFLOW LOCK" ลงไป
3. **Project Sources:** upload 5 ไฟล์ (HTML + 3 MD + xlsx)
4. กด Create

### ทุกครั้งที่เริ่มแชทใหม่ใน project
1. เปิด Grok Project COSMOS LOG → New chat
2. Paste บล็อก code ด้านบน (ตั้งแต่ "สวัสดีครับ" ถึง "section To-Do")
3. Grok จะอ่าน Project Sources + บล็อกนี้ + เริ่มทำงานต่อทันที

### ถ้าใช้ Claude/AI อื่น (ไม่มี Project Sources)
1. Paste บล็อก code ทั้งหมด
2. AI จะใช้ context จากบล็อก + ขอให้ครู paste ไฟล์ checkpoint
3. ทำต่อได้
