## 💡 Pro Tips

### 1. ลอง Level 1 ก่อนเสมอ
ประหยัดเวลา · MEMORY.md ส่วนใหญ่ทำงานได้

### 2. ระบุชื่อช็อตเฉพาะ
ดีกว่าบอกแค่ "ต่อ":
- ❌ "ต่อ"
- ✅ "ส่ง prompt EP03-MEERA-CARE ให้"

### 3. ถ้าตอบไม่ตรง · ส่ง correction
```
ลืม! workflow lock บอก "1 action ต่อช็อต"
แตก prompt นี้เป็น 2 sub-shots ใหม่
```

### 4. Pin ไฟล์ไว้
สร้าง shortcut/bookmark ของ:
- `JEN_RESUME_COMMAND.md` (อันนี้)
