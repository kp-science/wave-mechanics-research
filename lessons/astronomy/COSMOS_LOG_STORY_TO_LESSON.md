## 🧭 วิธีอ่านเอกสารนี้

แต่ละ Scene มี 5 ส่วน:

```
🎬 STORY BEAT       เรื่องที่เกิด · 1-2 ประโยค
❓ STUDENT'S NEED   คำถามที่นักเรียนเกิดขึ้นเอง
📚 CONCEPT          เนื้อหา ม.5 ที่ต้องรู้
🎮 ACTIVITY         กิจกรรมในระบบ
✨ RESOLUTION       เรื่องดำเนินต่อ + curriculum link
```

## 📐 Pattern Summary

| Story Element | Lesson Hook |
|---|---|
| **Problem** ที่ทีมต้องแก้ | สร้าง **need** ให้นักเรียนต้องการรู้ |
| **Mentor character** อธิบาย | ครูในเรื่อง · authoritative แต่ไม่ปรามาส |
| **VOID / Antagonist** | misconception ที่ต้องหักล้าง |
| **Father/family stakes** | emotional anchor · ผลของการเรียน |
| **Ship/Equipment** | tool ที่ใช้สมการจริง · v=H₀d, d=1/p, m-M=5log(d/10) |
| **Time pressure** | maintain engagement · pacing |
| **Reset/Preserve choice** | ethical dimension · critical thinking |

## 🔄 Story↔Lesson Loop ที่ใช้ทุก EP

```
1. Story sets crisis        ← cinematic / dialog / VOID
2. Need surfaces            ← นักเรียนรู้สึก "ต้องรู้"
3. Mentor introduces        ← ดร.ฮับเบิล / LEMAÎTRE
4. Activity teaches         ← lab / drag / calc / quiz
5. Apply solves crisis      ← boss / decision / progress
6. Story advances           ← reveal / hook / cliffhanger
7. Evidence collected       ← keyring / shard / log
8. Loop back                ← scene next
```
