## 4. ⭐ BOSS ใหม่ · "GENESIS FORGE" · Build the Universe

| EP | Boss | Pattern |
|---|---|---|

| EP06 | 15×15 grid maze | exploration |

| **EP08** | **Genesis Forge 4 epoch** | **construction · summative** |

### Concept
ผู้เล่น **สร้างเอกภพใหม่ทีละยุค** · VOID พยายามแทรก noise ทุก epoch
เลือกค่า/วัสดุ/เวลาให้ถูก · ผ่าน = epoch สำเร็จ · ครบ 4 = บิกแบงรอด

### 4 Epoch Phases

#### 🌌 Epoch 1 · INFLATION (10⁻³⁶ s)
- ตั้งค่า "Inflation factor" · ต้องขยาย 10²⁶ เท่า
- เลือก fundamental constants (gravity strength, Planck scale)
- ใช้ความรู้: EP04 quantum + EP01 fundamental forces
- VOID corrupt: ค่า gravity ผิด → universe collapse กลับ

#### 🔥 Epoch 2 · NUCLEOSYNTHESIS (3 นาที)
- ทำให้ proton + neutron → H, He, Li
- ตั้งอุณหภูมิ + ความหนาแน่น
- ใช้ความรู้: EP02 fusion + EP05 nuclear reaction
- VOID corrupt: อุณหภูมิเย็นเกิน → ไม่มีฟิวชัน

#### 💫 Epoch 3 · CMB & FIRST STARS (380,000 yr → 200 Myr)
- ปล่อย CMB ที่ 2.73 K
- จุด first stars · ทำให้ fusion เกิด
- ใช้ความรู้: EP04 CMB + EP01-03 stellar
- VOID corrupt: ลบ atom recombination → universe ยังขุ่น

#### 🌠 Epoch 4 · GALAXIES & STRUCTURE (1 Gyr+)
- จัดเรียง dark matter + ปกติ matter
- รวม first stars เป็น galaxies
- ใช้ความรู้: EP06 solar system + EP07 telescope (Hubble's law)
- VOID corrupt: scatter dark matter → ไม่มี galaxy formation

### Each Epoch Flow
```
1. VOID intro · ใส่ปัญหา + ข้อความ taunt
2. Player ใช้ "knowledge inventory" จาก S1 (4 EP × 4 OS = 16 items)
3. Forge UI · drag knowledge ไปใส่ epoch slot
4. Submit · stabilize universe หรือ collapse
5. Epoch ผ่าน → cinematic ยุคใหม่เกิด
```
