## 🔒 Workflow Lock v3 (อัปเดต 2026-05-10)

### 1. Format Prompt — Lean Pattern (ใช้กับทุก prompt)

```
สร้างวิดีโอ [N] วินาที · 16:9 · 480p ตัวอย่างความเร็วสูง

⭐ ตัวละครพูดประโยคนี้เท่านั้น (ห้ามเปลี่ยน · ห้ามแต่งเพิ่ม):
"[dialogue ภาษาไทย 100% ห้ามอักษรอังกฤษ]"

ภาษา: ภาษาไทยเท่านั้น · ห้ามภาษาอังกฤษ
ผู้พูด: [character + voice tone]
จำนวนคำ: 1 ประโยค · พูดครั้งเดียว · ไม่ซ้ำ · ไม่มีคำอื่น

สถานที่: [ภาษาไทย · ละเอียด · HUD เป็นไทย]
ตัวละคร: [Asian features ละเอียด · ผิว/โครงหน้า/ดวงตา/จมูก/ผม/ส่วนสูง/ชุด]

POSE: [pose override ระบุละเอียด · ห้าม pose ที่ไม่ต้องการ]
TIMELINE: [sync ภาพกับคำพูด · 0-N วิ ... N-M วิ ...]
CAMERA: [multi-technique · 3-5 stage]
LIGHT: [cinematic + emotional · cyan-only ถ้าไม่ต้องการทอง]
STYLE: cinematic photorealistic · 4K depth of field · [reference tone · ห้ามชื่อหนัง]

ห้าม:
- ภาษาอังกฤษในเสียงพูด
- ตัวละครพูดคำอื่นนอกเหนือประโยคที่กำหนด
- ตัวละครเป็นฝรั่ง คอเคเซียน ผิวขาวยุโรป ตาน้ำเงิน-ฟ้า-เขียว ผมบลอนด์
- รอยแผลบนตา · ข้ามตา · ในม่านตา · ดวงตาบาดเจ็บ
- ดวงตาเรืองแสง · ม่านตาเรืองแสง · iris glow
- ฝุ่นแสงสีทองที่ใบหน้า · god rays สีทอง (ใช้ cyan)
- pose คุกเข่า · นั่ง · ผีหลอน
- กล้อง static · มุมเดียว
- subtitle ซ้อน · อะนิเมะ · การ์ตูน
- ชื่อหนัง · ผู้กำกับ · franchise
- ใบหน้าตัวละครไม่ตรง reference image
```

### 2. Duration Rule
- **6 วินาที:** 1 character + 1 action / no character simple motion / 1 short dialogue
- **10 วินาที:** 1 character multi-action / dialogue ยาว / cinematic reveal / camera complex
- **อย่าใช้ 8 วิ** — Grok รองรับ 6 หรือ 10 วินาทีเท่านั้น

### 3. Camera Technique Library

| อารมณ์ | เทคนิคพิเศษ |
|---|---|
| Cosmic awe | God's eye crane down + dolly-in + lens compression |
| Mystery + revelation | Hitchcock dolly-zoom (vertigo) |
| Heroic command | Orbital arc 180-270° + low hero angle |
| Discovery | Push-in + crane up + rack focus |
| Cosmic majesty | Multi-axis sweep (pan + tilt + dolly) |
| Scientific beauty | 360° bullet-time orbit + vertical crane up |
| Tension | Crash zoom + Dutch tilt + handheld shake |
| Revelation | Pull-back reveal · CU → WIDE explosion |
| Quiet awe | Wide static · long take |
| Whisper / intimate | ECU + rack focus + slow drift |
| Captain peak ⭐ | 270° orbital + dolly-zoom + crash zoom + time ramp |
| Emotional cliffhanger | WIDE → tracking → 2-shot → rack focus → MED CU push-in |
| Multi-station tour | Whip-pan (cyan motion blur) ระหว่าง pilots |
| Match-cut interior→exterior | Frame composition center · scale match |

### 4. Logic Continuity Rules
- **Warp sequence:** Captain command interior → Pilots execute interior → WARP exterior → Warp Exit → Lonely Star
- **ยานเคลื่อนที่ ≠ คนยืนคุยกัน** (warp-exit ก่อนค่อยคุย)
- **ก่อน character dialogue** ต้องมี trigger (LEMAÎTRE alert / signal received)
- **ตัวละครหลายคนในเฟรม** → แตกเป็น sub-shots 1 ตัวต่อช็อต (whip-pan tour ได้)
- **Match-cut between scenes** — endframe scale + light tone ต้อง match กับ startframe ของช็อตถัดไป

## 🚫 Grok Limitations (อัปเดต 2026-05-10)

| ปัญหา | Workaround |
|---|---|
| Macro multi-objects (5+ ชิ้น) ใน 1 เฟรม | แตกเป็น sub-shots 1 ชิ้นต่อช็อต |
| Horror keywords trigger filter | ใช้ filter-safe wording (ethereal, swirling galaxy pattern) |
| ⭐ **Grok generate ฝรั่งแทนเอเชีย** | เขียน character description ละเอียด (ผิว/โครงหน้า/ดวงตา/จมูก) + ห้ามฝรั่งใน "ห้าม" |
| ⭐ **Dialogue เพี้ยนเมื่อมีอักษรอังกฤษปน** | Dialogue ภาษาไทย 100% (m → "แมกนิจูดที่เห็น" / d → "ระยะ") |
| ⭐ **Thai dialogue choppy/translation-style** | เขียนเป็น Thai natural · ไม่แปลตรง · ใช้สำนวน ("ไม้บรรทัดของจักรวาล" แทน "standard candle") |
| ⭐ **ตาเรืองแสงทอง/เปลวไฟ** | ระบุ "ดวงตาธรรมชาติ ไม่เรือง" + ใช้ cyan-only palette · ห้าม ECU ที่ตา |
| ⭐ **รอยแผลผิดตำแหน่ง (บนตา/ข้ามตา)** | ระบุระยะ "1.5 ซม. เหนือคิ้วขวา ห่างจากตา 2 ซม." + ห้าม scar 7 ตำแหน่ง |
| ⭐ **Franchise refs ติด filter** | ห้ามชื่อหนัง · ผู้กำกับ · ตัวละครหนัง |
| ⭐ **Voice-over English แทนไทย** | "ห้ามภาษาอังกฤษในเสียงพูด" + dialogue ที่ส่งให้ต้องไม่มีอักษรอังกฤษ |
| Character lip-sync ภาษาไทยไม่แม่น | ระบุ "พูดประโยคนี้เท่านั้น" + sync TTS ภายหลังถ้าจำเป็น |
| 5+ characters in frame หน้าหลุด | แตกเป็น 1 character ต่อช็อต / split-screen |
| Resolution: ใช้ Quality หมด rate limit | ใช้ **480p Speed** สำหรับ draft · Quality สำหรับ final shot |
