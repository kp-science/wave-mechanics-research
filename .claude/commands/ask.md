# /ask — ช่วยหาคำสั่งที่ต้องใช้

## งาน

### Phase 1: รับคำถาม
- ถ้าไม่มี argument → ถามผู้ใช้สั้น ๆ: "อยากทำอะไรครับ?"
- ถ้ามี argument → ใช้เป็น query เลย

### Phase 2: แนะนำ
1. **อ่าน frontmatter** ของไฟล์ `.claude/commands/*.md` ทุกไฟล์ (เฉพาะ 5 บรรทัดแรก · เร็ว · ไม่กิน token)
2. Match กับ query (keyword + description)
3. **แนะนำ 1-3 คำสั่ง** เรียง relevance
4. ต่อแต่ละคำสั่งให้:
   - ชื่อคำสั่ง + ตัวอย่างการใช้จริง
   - เหตุผลว่าทำไมตรง
   - ถ้ามี micro/macro ให้แนะนำว่าใช้ตัวไหนดี

## Output format
```
🎯 จาก "<query>" แนะนำ:

1️⃣ /phys-full waves 11 "<หัวข้อ>"
   → ใช้เมื่อ: สร้างแผนฟิสิกส์ครบวงจร 1 แผน
   → Token: ~15k

2️⃣ /phys-plan-new waves 11 "<หัวข้อ>"
   → ใช้เมื่อ: อยากทำทีละขั้น (ตรวจทานระหว่างทาง)
   → ต่อด้วย /phys-media-add · /phys-check

💡 Tip: ถ้ามาตรฐาน ใช้ #1 · ถ้าพิเศษ ใช้ #2
```

## ตัวอย่าง

**ผู้ใช้:**
```
/ask "อยากเพิ่มแผนฟิสิกส์"
```

**ผม:**
```
🎯 แนะนำ:
1️⃣ /phys-full waves <N> "<หัวข้อ>"  — เร็วสุด
2️⃣ /phys-plan-new + /phys-media-add — คุมคุณภาพ
```

**ผู้ใช้ (ไม่มี arg):**
```
/ask
```

**ผม:**
```
อยากทำอะไรครับ? เช่น:
  • "สร้างแผนใหม่"
  • "เอาบทเรียนเข้าเว็บ"
  • "เปิดวิชาใหม่"
  • "ดูว่าทำถึงไหน"
```

## Token budget
- อ่าน frontmatter ~5 บรรทัด × 38 ไฟล์ = ~2k tokens
- ไม่อ่านเนื้อคำสั่งเต็ม
- รายงาน ≤150 คำ
