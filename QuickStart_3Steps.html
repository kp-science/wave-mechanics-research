3.1 ลบโค้ดเดิม

3.2 เปิดไฟล์โค้ดจากผม

📋 เปิด AppsScript_Code.gs

ไฟล์จะเปิดในแท็บใหม่แสดงโค้ดเป็นข้อความ

3.3 คัดลอกโค้ดทั้งหมด

ในแท็บที่เปิดไฟล์ AppsScript_Code.gs:

  1. คลิกในพื้นที่แสดงโค้ด
  2. กด Ctrl+A — ข้อความทั้งหมดจะถูกไฮไลต์เป็นสีฟ้า
  3. กด Ctrl+C — คัดลอกเข้า clipboard
3.4 วางในหน้า Apps Script

สลับกลับไปแท็บ Apps Script Editor:

  1. คลิกในกล่องโค้ดที่ว่างเปล่า
  2. กด Ctrl+V — โค้ดควรปรากฏในกล่อง
✅ ตรวจสอบ: กล่องโค้ดต้องเริ่มต้นด้วย:
// ═══════════════════════════════════════════════════════════════════ // Research Data Collector + Analytics — วิจัยคลื่นกล // ครูโกเมน ปาปะโถ · โรงเรียนสตรีวิทยา · 2569 // ═══════════════════════════════════════════════════════════════════
และลงท้ายด้วย .setMimeType(ContentService.MimeType.JSON);
} โดยรวมมี ประมาณ 140 บรรทัด

วาง URL ลงเว็บแอป ⏱ 1 นาที

8.1 เปิดไฟล์เว็บแอปด้วย Text Editor

Mac: Finder → ไปที่โฟลเดอร์ วิจัย → คลิกขวาที่ วิจัยคลื่นกล_WebApp.html → Open With → TextEdit

Windows: File Explorer → คลิกขวา → Open with → Notepad

⚠️ อย่าดับเบิลคลิกธรรมดา ไฟล์จะเปิดในเบราว์เซอร์ ต้องเปิดด้วย Text Editor เท่านั้น
8.2 ค้นหาจุดที่ต้องแก้

กด Ctrl+F (หรือ Cmd+F บน Mac)

พิมพ์ข้อความค้นหา:

PASTE_YOUR_APPS_SCRIPT_URL_HERE

กด Enter → เคอร์เซอร์จะกระโดดไปที่บรรทัด:

const API_URL = 'PASTE_YOUR_APPS_SCRIPT_URL_HERE';
8.3 แทนที่ด้วย URL จริง

ลบข้อความ PASTE_YOUR_APPS_SCRIPT_URL_HERE ออก (เฉพาะข้อความ อย่าลบ เครื่องหมาย ' ทั้งสองข้าง)

วาง URL ที่คัดลอกมา:

ก่อนแก้
const API_URL = 'PASTE_YOUR_APPS_SCRIPT_URL_HERE';
หลังแก้ (ถูกต้อง)
const API_URL = 'https://script.google.com/macros/s/AKfycbx........../exec';
8.4 บันทึกไฟล์

กด Ctrl+S (Windows) หรือ Cmd+S (Mac)

ปิด Text Editor

✅ เช็คลิสต์หลังแก้ไข

  • URL เริ่มต้นด้วย https://script.google.com/macros/s/
  • URL ลงท้ายด้วย /exec
  • เครื่องหมาย ' (single quote) อยู่ครบทั้งซ้ายและขวาของ URL
  • มี ; (semicolon) ที่ท้ายบรรทัด
